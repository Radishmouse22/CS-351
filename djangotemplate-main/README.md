# Django Template
A standard template for developing Django applications.
This is what I use for my django applications as a template. It includes a standard homepage with placeholder images and nicely formated text.
It also has a contact page with a fully functioning cantact form built-in.

This project **does not require venv**. To use it, open a command promt with the folder open and type `python manage.py runserver 80`
This way you can type `127.0.0.1` in your **browser** to see the site.
